# 🌀 Helical Spring Optimization using SHMS Algorithm

## 📌 Overview
Innovation in engineering is no longer limited to designing better components—it is about designing smarter, data-driven systems.

This project focuses on optimizing the design of a **helical compression spring** using the **Snail Homing and Mating Search (SHMS) Algorithm**, a bio-inspired optimization technique.

The objective is to enhance performance, efficiency, and reliability through intelligent computational optimization.

---

## ⚙️ Problem Statement
Helical springs are widely used in:
- Automotive suspension systems
- Industrial machinery
- Mechanical assemblies

However, their design optimization is challenging due to:
- Multiple interacting variables  
- Non-linear relationships  
- Design trade-offs between strength, weight, and durability  

---

## 🧠 Approach

### 🔬 Bio-Inspired Optimization
We implemented the **Snail Homing and Mating Search (SHMS)** algorithm, which mimics:

- 🐌 **Homing Behavior** → Exploitation (refining optimal solutions)  
- 🐌 **Mating Behavior** → Exploration (searching new regions)  

This balance enables efficient optimization of complex engineering systems.

## 📊 Design Variables
Key parameters considered in the optimization:
- Wire diameter (d)  
- Mean coil diameter (D)  
- Number of active coils (N)  
- Material properties  
- Spring index

## 🎯 Objective
The optimization aims to:
- Minimize material usage / weight
